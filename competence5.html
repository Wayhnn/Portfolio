Description

Acquisition des compétences pour concevoir et mettre en œuvre des solutions de sécurité pour les systèmes d’information.
Maîtrise des outils de sécurité, des protocoles de sécurité et des techniques de maintenance.

OUTILS UTILISÉS

En plus de VS Code pour la programmation, nous avons utilisé Project Libre pour la partie gestion de projet.

LANGAGES

Dans le cadre de cette compétence, nous avons étudié les langages HTML, CSS et JavaScript

Le Site Tabas'KHO

Nous avons dévellopé un site WEB pour un client fictif. Celui-ci souhaitais que l'on réalise une refonte de son site de streaming. En partant de l'analyse de l'entretient du client, nous avons proposé une charte graphique ainsi que différentes pages Web répondant à ses besoins.
